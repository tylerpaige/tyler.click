Unix time refers to the number of seconds that have elapsed since midnight on January 1, 1970, coordinated universal time.

Stay counting.

 

Unix time is used in programming to evaluate an order of events. It is not a standard for time like a 12- or 24-hour clock. Its only concerns are which events happened first, and how many seconds (arguably the only timescale humans have an understanding of) passed in-between the events.

Stay counting.
